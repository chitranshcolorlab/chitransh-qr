Upload:
1. Extract this ZIP.
2. Open https://github.com/chitranshcolorlab/chitransh-qr
3. Click Add file > Upload files.
4. Select ALL extracted files and upload them to the repository ROOT.
5. Choose Commit directly to the main branch.
6. Click Commit changes.
7. Wait 1-3 minutes, then open:
   https://chitranshcolorlab.github.io/chitransh-qr/
8. Press Ctrl + F5 once if the old version appears.
